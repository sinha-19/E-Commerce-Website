## Getting Started

### Prerequisites

- Node.js (version X.X.X or above)
- npm or yarn

### Installation

```bash
git clone https://github.com/sinha-19/E-Commerce-Website.git
cd E-Commerce-Website
npm install
# or
yarn install
```

### Running the Application

```bash
npm run start
# or
yarn start
```
